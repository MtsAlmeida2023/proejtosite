# Projeto integrador

Esse é um projeto para a matéria de Projeto Integrador dos alunos do Eixo Computação da UNIVESP.

## Projeto API Web "Caotrole": Aplicação desenvolvida para ONG de cães em Itatiba, com o intuito de melhorar o controle de cães e setores no estabelecimento. Apresentação do projeto em https://youtu.be/B-iAYsPKX1g

### Features
- Login
- Autenticação de login
- Armazenamento criptografado de senhas
- Cadastro de animais para doação
- Fotos e descrição dos animais
- Informações pertinentes a adoção
